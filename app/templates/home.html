NHS Hospital Activity Explorer

This web application analyses NHS hospital activity data collected between April 2020 and March 2021.

Application Features

  • Browse NHS healthcare providers across England
  • Search and filter providers by region and activity
  • Analyse admissions, outpatient attendance, and DNA appointments
  • Compare up to three providers side-by-side
  • Explore NHS open healthcare datasets

Dataset Information

The application uses provisional NHS Hospital Episode Statistics (HES) data from England covering:

  • Emergency admissions
  • Elective admissions
  • Outpatient attendance
  • DNA (Did Not Attend) appointments

Quick Navigation

  • Browse NHS Providers
  • View Analytics Dashboard
  • Compare Providers
